# SAMPA配線

ここではSAMPAチップの配線のめものまとめておきます。

SAMIDAREには4つのSAMPAチップが搭載されており、SAMPA0からSAMPA3と呼ばれています。ここではSAMPA0の配線をまとめておきます。

ただし、`SIG`と`SOP`、`SON`はSAMPA`i`に対して`i+1`になっています。

### SOPとSONの意味のついて

SAMPAの出力である`SOP`と`SON`はそれぞれPositiveとNegativeの意味で、差動信号になっています。

10bitのデータが出力されるため、`SOP[10:0]`と`SON[10:0]`の2つの信号で1つのデータを表現しています。

各bitが出力する信号は以下の通りです。

| :--------: | :-------------------: | :-----: |

| `AB20`     | `SDO3+`               | `SOP9`  |
